CentOS7.4安装svn服务器

1:安装svn服务器端
yum install subversion

2:创建svn工程文件夹
mkdir -p /var/svn/SIE
svnadmin create /var/svn/SIE

3:修改配置文件
ls /var/svn/SIE/conf

vi svnserve.conf
auth-access = write
password-db = passwd
authz-db = authz
realm = SIE

vi passwd
root = root123

vi authz
[/]
root = rw
* =

4:启动服务
svnserve -d -r /var/svn

5:安装svn客户端checkout
svn://ip:3690/SIE
