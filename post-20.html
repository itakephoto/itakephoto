阿里云上Apache多站点绑定


1:修改配置文件
vi /etc/httpd/conf/httpd.conf

<VirtualHost *:80>
ServerName lijiahong.com
DocumentRoot /var/www/html/emlog/
<Directory "/var/www/html/emlog">
Options Indexes FollowSymLinks
AllowOverride None
Order allow,deny
Allow from all
</Directory>
</VirtualHost> 

<VirtualHost *:80>
ServerName ivylee.cn
DocumentRoot /var/www/html/ivy/
<Directory "/var/www/html/ivy">
Options Indexes FollowSymLinks
AllowOverride None
Order allow,deny
Allow from all
</Directory>
</VirtualHost>

<VirtualHost *:80>
ServerName liaifei.com
DocumentRoot /var/www/html/ivy/
<Directory "/var/www/html/ivy">
Options Indexes FollowSymLinks
AllowOverride None
Order allow,deny
Allow from all
</Directory>
</VirtualHost>

2:重启apache
/etc/init.d/httpd restart
