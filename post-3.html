Linux安装JDK

1）把安装文件文件保存在/usr/java/下 ,并切换到root用户 
#> mkdir /usr/java
#> cp jdk-6u22-linux-i586.bin /usr/java/
#> su - root
#> 输入root密码

2）添加可执行的权限
#> chmod 755 jdk-6u22-linux-i586.bin 或 chmod a+x jdk-6u22-linu

3) 安装jdk1.6
#> ./jdk-6u22-linux-i586.bin #(大约9页 License) say yes to install jdk
完成后 /usr/java目录下会新建一个名“jdk1.6.0_22”的目录，至此已经安装好了新的jdk

4) 配置jdk1.6.0_22环境变量
查看 确认新安装的jdk1.6.0_22 的目录为 /usr/java/default
配置jdk1.6.0_22环境变量
#> vi /etc/profile
export JAVA_HOME=/usr/java/default
export CLASSPATH=.:$JAVA_HOME/lib:$JAVA_HOME/jre/lib
export PATH=$JAVA_HOME/lib:$JAVA_HOME/bin:$JAVA_HOME/jre/bin:$PATH:$HOME/bin:$PATH
export JAVA_TOOLS=$JAVA_HOME/lib/tools.jar

5) 使环境变量生效：
#> source /etc/profile
#> java -version
